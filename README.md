# Blackjack

Description
Simple Blackjack game where you can create and join dealers with other players and play in real time

After creating a dealer and starting a game:

You can hit to get new cards or stand to await other players

Then when all stand (or busts) victory is evaluated and a new game can be started

Instructions

Game has only been run in development
Clone the repository
Run api with dotnet run
Run app with dotnet run

Visit localhost:5016 to play the game
